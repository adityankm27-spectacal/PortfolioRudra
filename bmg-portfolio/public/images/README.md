# Images

Drop image files here and the site will use them automatically (it shows a
styled placeholder until the real file exists).

## Hero photo (required)

- **`profile.png`** — your cutout photo for the hero (the figure in front of the
  giant name). A **transparent-background PNG** looks best (like the reference).
  If you only have a photo with a background, drop it and ask Claude to remove
  the background for you.

## Project screenshots (optional)

Put build screenshots in `work/` using these exact filenames (defined in
`src/lib/content.ts`):

- `work/sci-fi-lab.jpg`
- `work/hobbit-home.jpg`
- `work/waterfall.jpg`
- `work/park-entrance.jpg`
- `work/military-outpost.jpg`
- `work/misty-forest.jpg`

Until added, each project card shows a colored gradient placeholder.
